# Extensions

Purpose-built tools and automation for specific security workflows.

## Overview

Extensions provide advanced functionality for common security operations tasks:

- Automated artifact collection
- YARA rule management
- Cloud resource monitoring
- Configuration synchronization
- And much more...

## LimaCharlie Extensions

Official extensions built and maintained by LimaCharlie:

- [Artifact Extension](LimaCharlie_Extensions/ext-artifact.md) - Automated evidence collection
- [YARA Manager](LimaCharlie_Extensions/ext-yara-manager.md) - Deploy and manage YARA rules
- [Git Sync](LimaCharlie_Extensions/ext-git-sync.md) - Version control for configurations
- [Infrastructure](LimaCharlie_Extensions/ext-infrastructure.md) - Cloud resource monitoring
- [Exfil Detection](LimaCharlie_Extensions/ext-exfil.md) - Data exfiltration monitoring
- [Integrity Monitoring](LimaCharlie_Extensions/ext-integrity.md) - File integrity monitoring
- [EPP](Limacharlie_Extensions/ext-epp.md) - Endpoint protection platform
- [Dumper](LimaCharlie_Extensions/ext-dumper.md) - Memory dumping automation
