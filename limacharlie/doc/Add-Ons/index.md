# Add-Ons

Expand LimaCharlie capabilities with API integrations, extensions, and third-party tools.

## Overview

Add-Ons extend the LimaCharlie platform with additional functionality:

- **API Integrations**: Connect to threat intelligence, enrichment, and security services
- **Extensions**: Purpose-built tools for specific security workflows
- **Services**: Advanced platform capabilities

## Categories

### [API Integrations](API_Integrations/index.md)

Connect to external services for enrichment and analysis:

- VirusTotal - File and URL reputation
- GreyNoise - IP reputation and context
- Hybrid Analysis - Malware sandboxing
- IP Geolocation - Geographic context
- And many more...

### [Extensions](Extensions/index.md)

Purpose-built tools for specific workflows:

- Artifact Collection - Automated evidence gathering
- YARA Manager - Deploy and manage YARA rules
- Git Sync - Version control for configurations
- Infrastructure - Cloud resource monitoring
- And many more...

## Using Add-Ons

1. Browse available add-ons in the [web console](https://app.limacharlie.io)
2. Review documentation and requirements
3. Configure credentials and settings
4. Test the integration
5. Incorporate into detection rules and workflows

See the [Building Extensions](Extensions/Building/building-extensions.md) documentation to create custom add-ons for your organization.
