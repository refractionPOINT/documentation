# Getting Started with LimaCharlie

Welcome to LimaCharlie! This section will help you understand the platform and get started quickly.

## Overview

LimaCharlie is the SecOps Cloud Platform that delivers comprehensive enterprise protection by bringing together critical cybersecurity capabilities in a unified, cloud-native platform.

## Key Concepts

Understanding these core concepts will help you make the most of LimaCharlie:

- **Organizations**: Your security workspace and billing unit
- **Sensors**: Agents deployed on endpoints to collect telemetry
- **Detection & Response Rules**: Custom logic for threat detection and automated response
- **Outputs**: Destinations for your security telemetry (SIEM, webhooks, etc.)
- **Add-Ons**: Extensions and integrations to expand platform capabilities
