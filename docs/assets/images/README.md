# Logo and Image Assets

## Required Assets

This directory should contain the official LimaCharlie branding assets:

### Logo Files

1. **logo.svg** - Main logo for dark backgrounds
   - Download from: https://brandfetch.com/limacharlie.io
   - Or use the official logo from LimaCharlie marketing assets
   - Current file is a placeholder - replace with official logo

2. **logo-dark.svg** (optional) - Logo variant for light backgrounds
   - Use if you have a specific light background variant

3. **favicon.png** - Site favicon
   - Recommended size: 512x512px or 256x256px
   - Format: PNG with transparency
   - Download icon from: https://brandfetch.com/limacharlie.io
   - Current file is a placeholder - replace with official favicon

## Getting Official Assets

Official LimaCharlie brand assets can be obtained from:

1. **Brandfetch**: https://brandfetch.com/limacharlie.io
   - Logos in SVG and PNG formats
   - Official brand colors
   - Typography information

2. **LimaCharlie Marketing Team**
   - Contact marketing@limacharlie.io for official brand assets
   - Request high-resolution logos and brand guidelines

3. **Website Assets**
   - Some assets may be available on https://limacharlie.io

## Brand Guidelines

When replacing the placeholder assets, ensure they match:

- **Primary Color**: Midnight (#00183c)
- **Accent Color**: Maroon Flush (#c32152)
- **Fonts**: Syne (headings), Rubik (body text)

## File Requirements

- **Format**: SVG preferred for logo (scales perfectly at any size)
- **Background**: Transparent or appropriate for dark theme
- **Size**: Logo should be recognizable at small sizes (navbar display)
- **Colors**: Should work with the dark theme (#00183c background)
