# Platform Management

Configure and manage your LimaCharlie organization, users, and platform settings.

## Overview

Platform management encompasses:

- Organization configuration
- User access control
- API key management
- Billing and usage
- Audit logging
- Integration settings

## Key Areas

### Organization Management
- Create and configure organizations
- Manage organization settings
- Configure retention policies
- Set up organization secrets

### User & Access Control
- Invite team members
- Configure role-based access control (RBAC)
- Manage API keys
- Set up SSO (Single Sign-On)

### Billing & Usage
- Monitor sensor count
- Track data ingestion
- View usage reports
- Manage subscriptions

### Audit & Compliance
- Review audit logs
- Export compliance reports
- Monitor administrative actions

## Best Practices

- Use separate organizations for different environments (dev, staging, production)
- Implement least-privilege access control
- Rotate API keys regularly
- Monitor audit logs for suspicious activity
- Set up usage alerts for cost control
