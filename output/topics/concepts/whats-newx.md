## Detection Categories

The Proactive Detections service includes comprehensive coverage across multiple threat categories:

### Malicious PowerShell
Detections targeting PowerShell-based attacks including encoded commands, fileless execution, download cradles, obfuscated scripts, suspicious commands, Empire Framework usage, Mimikatz execution, Invoke-Expression abuse, Base64 encoding, and hidden window execution.

### Suspicious Registry Modifications
Detections for registry changes related to credential access, persistence mechanisms, and defense evasion techniques.

### Suspicious Process Injection
Detections covering various process injection techniques including process hollowing, reflective DLL injection, APC injection, remote thread creation, process doppelganging, AtomBombing, and thread execution hijacking.

### Suspicious Network Activity
Detections identifying C2 communication patterns, DNS tunneling, unusual outbound connections, port scanning, and beaconing behavior.

### Suspicious File Operations
Detections for ransomware indicators, file deletion, file encryption, mass file modifications, and unusual file extensions.

### Suspicious Scheduled Tasks
Detections for hidden tasks, unusual execution times, privilege escalation attempts, and persistence mechanisms.

### Credential Dumping
Detections covering LSASS access, registry hive export, SAM database access, DCSync attacks, and password filter DLL abuse.

### Lateral Movement
Detections for remote service creation, Remote Desktop Protocol usage, Windows Admin Shares access, PsExec usage, and WMI command execution.

### Privilege Escalation
Detections targeting token manipulation, UAC bypass, exploitation attempts, access token manipulation, and process injection for privilege escalation.

### Defense Evasion
Detections for indicator removal and masquerading techniques.
